# Pepita

<img src="img/pepita.jpg" height="150" width="150">

*Pepita* es una golondrina que sabe volar y comer.

* Al volar gasta 1 joule de energía por cada kilómetro recorrido
* Al comer adquiere la energia del alimento que ingiere
* Además, si gasta mucha energía, Pepita queda cansada

## Probar el programa

Levantar una consola REPL con el archivo [pepita.wlk](pepita.wlk) y probar el programa

* Ver que pepita **no** empieza cansada
* Hacerla volar hasta que se canse
* Dale de comer ambas comidas y ver cómo cambia su energía
* Madurar la manzana y ver cómo aporta más energía

## Extender el programa

### Agregar una nueva comida

Queremos extender el programa para que Pepita coma una nueva comida

* ¿Qué modificaciones habría que hacer?
* Inventá una comida nueva y hacé que Pepita la coma
* ¿Qué mensaje(s) debe entender la nueva comida? ¿Por qué?

### Cambiar una comida existente

Hacer que la _manzana_

* Al llegar al grado de madurez `5` se reinicie a `1`
* Poder reiniciar la madurez de la manzana cuando queramos (enviándole un mensaje), independientemente del grado actual
